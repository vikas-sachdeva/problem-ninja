Given an array of distinct integers(considering only positive numbers) and a number ‘m’, find the number of triplets with product equal to ‘m’.

**Example - 1**:

**Input**: 
    
    arr[] = { 1, 4, 6, 2, 3, 8}  
    m = 24

**Output** : 

    3
    {1, 4, 6} {1, 3, 8} {4, 2, 3}

**Example - 2**:

**Input**: 

    arr[] = { 0, 4, 6, 2, 3, 8}  
    m = 18

**Output** :

    0
